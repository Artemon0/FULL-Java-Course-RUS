# 🎮 Текстовая RPG Игра - Финальный проект Beginner

## 📋 Описание

Полноценная текстовая RPG игра с:
- ✅ Системой персонажей (3 класса)
- ✅ Боевой системой
- ✅ Системой опыта и уровней
- ✅ Инвентарём
- ✅ Врагами разных уровней
- ✅ Предметами и зельями

## 📁 Структура проекта

```
project-text-rpg/
├── Character.java       - Базовый абстрактный класс персонажа
├── Player.java          - Класс игрока с прогрессией
├── Enemy.java           - Класс врага
├── Item.java            - Базовый класс предмета + HealthPotion, Weapon
├── Inventory.java       - Система инвентаря
├── CombatSystem.java    - Боевая система
└── Game.java            - Главный класс игры
```

## 🚀 Компиляция и запуск

```cmd
cd beginner\project-text-rpg
```


## 🎮 Как играть

1. **Создание персонажа:**
   - Введите имя героя
   - Выберите класс (Воин/Маг/Лучник)

2. **Меню игры:**
   - `1` - Исследовать (встреча с врагами или сокровища)
   - `2` - Инвентарь (просмотр предметов)
   - `3` - Характеристики (статистика персонажа)
   - `4` - Отдохнуть (восстановить здоровье)
   - `0` - Выход

3. **Бой:**
   - `1` - Атаковать врага
   - `2` - Использовать предмет
   - `3` - Бежать из боя

- Особенность: Высокая выживаемость

### Маг
- HP: 80
- MP: 150
- Атака: 20
- Защита: 5
- Особенность: Высокий урон

### Лучник
- HP: 100
- MP: 80
- Атака: 18
- Защита: 7
- Особенность: Баланс

## 📊 Система прогрессии

- Повышайте уровень для улучшения характеристик
- Собирайте золото для покупок
- Находите зелья здоровья

## 🔧 Технические детали

### Использованные концепции ООП:
- ✅ Абстрактные классы (Character)
- ✅ Наследование (Player, Enemy extends Character)
- ✅ Полиморфизм (Item и его подклассы)
- ✅ Инкапсуляция (private поля, public методы)
- ✅ Композиция (Player содержит Inventory)

### Паттерны проектирования:
- Template Method (Character)
- Strategy (Item.use())
- Facade (CombatSystem)
